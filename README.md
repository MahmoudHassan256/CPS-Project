## Getting Started

### Prerequisites

- Java 8 or higher installed
- Maven/Gradle

### Build & Run

#### Using Maven

```bash
mvn clean install
java -jar target/cps-project.jar
```

#### Using Gradle

```bash
./gradlew build
java -jar build/libs/cps-project.jar
```

#### From Sources

1. Clone the repository:
   ```bash
   git clone https://github.com/MahmoudHassan256/CPS-Project.git
   cd CPS-Project
   ```

2. Build and run as above.
